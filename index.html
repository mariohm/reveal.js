Mi primera aplicación Node.js

							
Instalar Node.js mediante el gestor de paquetes de Node: npm
$ sudo apt-get install npm
Creamos directorio de aplicación y entramos en él
$ mkdir node_app && cd node_app
Instalamos, mediante npm, paquetes previos
$ npm install express && npm install jade
Creamos esqueleto de nuestra aplicación gracias a Express
$ ./node_modules/express/bin/express
Instalamos dependencias
$ npm install
Lanzamos nuestra primera aplicación Node.js
$ node app.js
